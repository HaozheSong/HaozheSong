High-performance Dynamic Memory Allocator

C implementation of malloc, calloc, realloc and free functions with a high average throughput of 10804 Kops/sec and a high average memory utilization of 74.2%.

Achieved by segregated free lists which are doubly linked lists to store information about free memory blocks based on the block size.

Increased utilization by 15% using 16-byte mini blocks and footerless allocated memory blocks

Utilized better fit algorithm to search for the wanted block in segregated free lists, a sweet spot between throughput and utilization.

Multithreading Caching Web Proxy

A simplified web proxy in C to proxy and cache HTTP 1.0 GET requests with multithreading.

Network programming using C socket interface and RIO (robust I/O) package.

Served multiple simultaneous requests by POSIX threads and ensured thread safety by mutex lock.

Cache server response in 1 MB software memory cache based on LRU strategy.
